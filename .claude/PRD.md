# Product Requirements Document (PRD)

> **Instructions for new projects:**  
> Delete everything below this line and replace it with your project's PRD.  
> To generate a PRD from your idea, tell your AI assistant what you're building  
> and run: `/create-prd`
>
> The AI will ask clarifying questions and generate a complete PRD covering:  
> Executive Summary, Mission, Target Users, MVP Scope, User Stories,  
> Architecture, Tech Stack, Security, API Spec, Success Criteria, and Phases.

---

## How to Fill This In

### Option A — Let the AI write it (recommended)
1. Open chat with your AI assistant
2. Describe your app idea in 2-3 sentences
3. Run `/create-prd`
4. The AI will overwrite this file with a complete PRD

### Option B — Write it manually
Use the sections below as a guide. Delete sections that don't apply.
